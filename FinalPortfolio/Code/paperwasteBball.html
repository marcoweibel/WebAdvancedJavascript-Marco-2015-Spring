Paper Waste B-Ball Recycling Bin

A Collab w/ Enayet | Kate | Juno


A collaborative project with Enayet Kabir, Kate Wallace & Shiqi Shen (Juno).

A project to encourage recycling of paper in a public school space. We made an interactive trash bin that made people subconciously recycle paper more and have fun while doing so. Using 2 Arduino's, multiple flex sensors, a piezo speaker and a whole lotta wires, we created a basketball/skeeball inspired reycling bin that had different scoring sections for different point values.

The aim of the game was to play "Paperwaste B-Ball" against your friends and see who could rack up the highest score. The backboard of the recycling bin had an LED display that would reflect the current players score. We also implemented a reset button so that you could reset the score for new challengers.

The scoring sections was laser cut plywood, and we used transparent plastic colored sheets attached to flex sensors to create the scoring mechanism. Each time a player scored, the appropriate score would be updated and a specific sound would be played from the piezo speaker.
