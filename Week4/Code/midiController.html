MIDI Controller

Fabricated a MIDI controller by using an Arduino Leonardo as well as some parts from adafruit. The wooden case was laser cut and parts were all sourced from adafruit.


The controller works as true MIDI, the leonardo chip was converted into a teensy, thus it connects thru MIDI and not serial like most arduinos.
