Live Visual Rig

Created using processing as well as various libraries, this project aimed to create a modular, Live Visual Rig to aid musicians/artist in performance.


Connections between a MIDI controller were mapped to different classes that drew P3D shapes as well as particle systems. Also, a kinect was used to map out the musicians/artist body so as to be able to project directly onto them.
